### What Was Learned
- LiteSpeed caching on this server can take 3-5 minutes to invalidate
- Touching .htaccess (adding a comment) triggers cache purge
- The "Proclaiming Truth. Impacting Lives" tagline was PART OF THE REWRITE, not the original theme
- Original theme uses: "Church of Christ / K.L. Rao Nagar" as hero heading, "Evangelist • Bible Teacher • Church Planter" as credentials
- LiteSpeed page cache is very aggressive. Touching .htaccess doesn't always clear it. Use `?nocache=` query string to verify changes are live.
- header.php inline `<style>` overrides external style.css (same specificity, later in DOM). Always sync both.
- `pre_get_document_title` filter is the correct WordPress 7.0 hook to override title
- LiteSpeed Cache plugin can be purged via PHP: `do_action('litespeed_purge_all')` + `wp_cache_flush()`
