### Navbar Centering Fix (continued from earlier session)
- Root cause: `position: fixed; left: 0; right: 0` on `.fm-navbar` computed to width=1265px (viewport 1280px - 15px scrollbar). Container centered within 1265px instead of full viewport, causing 7.5px offset.
- Fix: Changed `right: 0` to `width: 100vw` + `overflow-x: hidden` on `.fm-navbar`. This makes the navbar span the full viewport width (1280px) so `margin: 0 auto` on the container centers at true viewport center.
- Result: Content center now exactly 640px = viewport center (0px offset, previously 7.5px)
- Also consolidated duplicate `.fm-navbar` rules into one clean declaration
- File: /home/jay/faithful-messenger-theme/style.css
